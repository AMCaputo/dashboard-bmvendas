## CONTEXTO
na pasta tem uma base de dados de vendas da empresa B Mussungo & Filhos Comércio Geral, Lda. 
A base de dados retrata as vendas das quatro lojas desta empresa. 

## OBJETIVO
Crie um dashbord interativo completo usando python e streamlit

## KPIs principais(cards, no topo)
- Total gerais das vendas por ano
- Total geral das vendas, por loja

Os graficos para comparar as vendas anuais devem ser de barra e para as lojas de piza ou rosca

cada card deve mostrar a variação dos peridios anteriores
botões de filtros para seleção de ano, mensal, trimestral e bisexto para se acompanhar 
a evolução das vendas de cada ano e cada loja.
